# Typografia Mimetyczna: Ut pictura poesis

> "Tekst nie leży na obrazie. Tekst NAŚLADUJE materię."

Eksperymentalny projekt eksplorujący wcielenie poezji w fakturę wizualną przy użyciu AI.

---

## 🎯 Manifest

1. **Mimesis, nie ilustracja** – litery wcielają się w kamień, wodę, tkaninę, drewno
2. **Partnerstwo, nie posłuszeństwo** – AI jest mimetykiem wizualnym, nie narzędziem
3. **Funkcjonalność, nie ozdobność** – obraz UZUPEŁNIA tekst, nie konkuruje z nim

---

## 🚀 Szybki start

1. Przeczytaj `01_METODA/Biblia_v3.6.md`
2. Skopiuj szablon z `04_PROMPTY/prompt_template.txt`
3. Dostosuj do swojego tekstu + dodaj `prompt_lock.txt`
4. Generuj w Groku z `passive_mode.txt` aktywowanym
5. Analizuj: czy tekst JEST fakturą, nie nakładką?

[Pełna instrukcja → `02_START.md`](02_START.md)
